# jsbox-scripts

A collection of useful scripts for JSBox.

## Auto build artifacts (.box)

GitHub Actions will automatically package each script directory into `.box` files and publish them to the `release` branch (excluding `EasyJsBox`).

### Install via URL Scheme

- BoxJsHelper  
  - Artifact: https://raw.githubusercontent.com/ipuppet/jsbox-scripts/release/BoxJsHelper.box  
  - Install: jsbox://import?url=https%3A%2F%2Fraw.githubusercontent.com%2Fipuppet%2Fjsbox-scripts%2Frelease%2FBoxJsHelper.box&name=BoxJsHelper
